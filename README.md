# Soil Carbon Estimation – ML Experiment

This repository contains a small machine learning experiment related to
Soil Organic Carbon (SOC) estimation using Earth Observation data.

The work explores preprocessing geospatial datasets and analyzing
relationships between biomass patterns and SOC measurements.

## Methods
- Python
- Pandas
- NumPy
- Scikit-learn
- Geospatial processing

## Tasks
- Process SOC raster data
- Align Above Ground Biomass (AGB) datasets
- Apply clustering to identify biomass regimes
- Analyze correlation between clusters and SOC
